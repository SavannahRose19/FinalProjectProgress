HexMilkTea’s Café – CSC104 Checkpoint

This website is a checkpoint version of my final project. It is based on my streaming brand, HexMilkTea’s Café, and uses a consistent cozy horror theme across all pages.

Pages:

index.html  
The homepage includes the banner, logo, navigation, and a welcome section that introduces the vibe of the site and community.

about.html  
This page explains what HexMilkTea’s Café is and what viewers can expect from the stream and overall theme.

games.html  
This page includes a table that lists the types of games streamed and describes the vibe of each.

css/styles.css  
This external stylesheet controls the layout, colors, navigation styling, and overall design of the website.

All files are organized into folders (css and images), and the site uses external CSS for consistent design.
